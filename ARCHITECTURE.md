## Key Features Summary

1. **Role-Based Access Control**
   - Owner can create tasks
   - Members can view all but only update own

2. **Team Context**
   - Tasks linked to teams via `for_team` field
   - Only team members can be assigned

3. **Status Management**
   - Simple workflow: pending â†’ in-progress â†’ completed
   - Self-service for assigned members

4. **Real-Time Updates**
   - UI refreshes after actions
   - Activity logging for audit trail

5. **User Experience**
   - Sidebar shows all members
   - Click member to filter tasks
   - Visual status indicators
   - One-click status updates
